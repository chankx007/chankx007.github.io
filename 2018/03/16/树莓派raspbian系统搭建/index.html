树莓派--raspbian系统搭建

发表于 2018-03-16 | 分类于 树莓派

材料/工具

1.硬件

  • Raspberry Pi 3主板
  • TF内存卡(推荐16G)
  • 读卡器
  • 网线
  • HDMI线和支持HDMI的显示器(非必要)

2.软件

  • Win32DiskImager用于向内存卡写入系统，点击下载
  • 树莓派官方系统Raspbian，点击下载

系统搭建

Raspbian官方系统下载

打开树莓派官方网站的下载页面，选择RASPBIAN，官方提供两种下载，第一种DESKTOP版本是带有操作界面的，下载后解压系统有4G大，因此内存卡容量太小，新手推荐使用；第二种LITE版本，即使简化版，没有操作界面。

系统写入

系统下载后解压，打开Win32DiskImager.exe工具。选中img文件，点击Write按钮，向内存卡写入系统，等待进度条完成，这就将系统写进了内存卡，最后将内存卡插进树莓派并通电

远程登录

假设我们手头上没有支持HDMI的显示器，我们还可以通过远程桌面进入到树莓派系统，不过首先要获取树莓派的IP地址。

  • 获取IP地址，将树莓派通过网线和路由器连接到一起，进入路由器管理页面，查看DHCP列表，查找树莓派的IP
  • 得到树莓派的IP地址后，在电脑命令行下运行mstsc命令，打开远程桌面登录
  • 进入到树莓派系统时候需要账号密码，官方系统默认的登录帐号为 pi 密码是 raspberry
  • 登录后可以在系统配置里修改系统语言和更改登录密码。到这里系统搭建基本已经完成，官方完整版系统里面带有Python2.7和Python3，例外还有java编辑器等，习惯使用图形界面的小伙伴，使用完整版来学习树莓派是非常方便的。
    最后奉上登陆后的操作界面

    结束
